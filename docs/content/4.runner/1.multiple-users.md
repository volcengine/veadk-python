## 定义

在程序中，为对外提供多租户能力，您通常需要在代码中设计三个主要属性：

::field-group
  ::field{name="app_name" type="string"}
    对外提供服务的名称
  ::

  ::field{name="user_id" type="string"}
    用户唯一标识符
  ::

  ::field{name="session_id" type="string"}
    会话唯一标识符
  ::
::

## 资源划分对照

| 资源 | 索引划分 |
| :- | :- |
| 短期记忆 | `app_name` `user_id` `session_id` |
| 长期记忆 | `app_name` `user_id` |
| 知识库 | `app_name` |
