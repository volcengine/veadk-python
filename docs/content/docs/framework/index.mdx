**VeADK**（Volcengine Agent Development Kit）是火山引擎的智能体开发框架。它把模型、工具、记忆、知识库等能力组合成可复用的组件，并与火山引擎生态深度集成，让你用更少的代码构建生产级 AI 智能体。

```python
import asyncio
from veadk import Agent, Runner

agent = Agent(instruction="你是一个有帮助的助手。")
print(asyncio.run(Runner(agent=agent).run("你好，介绍下你自己")))
```
