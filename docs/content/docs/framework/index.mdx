火山引擎智能体框架 **VeADK（Volcengine Agent Development Kit）** 由**火山引擎**推出，为 Agent 智能体应用的构建提供开发、部署、观测、评测等全流程云原生解决方案。相较于现有的智能体开发框架，VeADK 与火山引擎产品体系深度融合，帮助开发者更高效地构建企业级 AI 智能体应用。
