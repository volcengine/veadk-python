## 前置条件

- Python `3.10` 或更高版本
- 一个火山方舟（Volcengine Ark）平台的 API Key

## 安装 VeADK

你可以通过 pip、源码或官方镜像三种方式安装 VeADK。

<Tabs items={['pip', '源码', '镜像']}>
  <Tab value="pip">
    ```bash
    # 稳定版
    pip install -U veadk-python
    ```

    安装包含知识库等可选能力的扩展包：

    ```bash
    pip install -U "veadk-python[extensions]"
    ```

    安装完成后，可通过以下命令验证：

    ```bash
    veadk --version
    ```
  </Tab>
  <Tab value="源码">
    安装主分支预览版：

    ```bash
    pip install git+https://github.com/volcengine/veadk-python.git@main
    ```

    安装指定分支：

    ```bash
    pip install git+https://github.com/volcengine/veadk-python.git@[branch-name]
    ```
  </Tab>
  <Tab value="镜像">
    VeADK 提供了 Python 版的镜像仓库，可直接用作基础镜像：

    ```bash
    # 稳定版
    veadk-cn-beijing.cr.volces.com/veadk/veadk-python:latest

    # 主分支预览版
    veadk-cn-beijing.cr.volces.com/veadk/veadk-python:preview

    # 指定版本号
    veadk-cn-beijing.cr.volces.com/veadk/veadk-python:<version>
    ```
  </Tab>
</Tabs>

<Callout type="info">
  如果你更习惯使用 `uv`，可以执行 `uv venv --python 3.10 && source .venv/bin/activate`，随后使用 `uv add veadk-python`（或 `uv add "veadk-python[extensions]"`）进行安装。
</Callout>

## 配置 API Key

VeADK 默认使用火山方舟平台的模型。安装完成后，需设置 `MODEL_AGENT_API_KEY` 环境变量为你的火山方舟 API Key：

```bash
export MODEL_AGENT_API_KEY="your-ark-api-key"
```

你也可以将其写入项目根目录下的 `.env` 文件：

```bash title=".env"
MODEL_AGENT_API_KEY=your-ark-api-key
```

<Callout type="info">
  关于配置项的完整说明（环境变量、`config.yaml`、内置常量及优先级），请参阅[配置](/cn/docs/framework/configuration)。
</Callout>
