## 前置条件

在开始之前，你需要安装 VeADK，并设置 `MODEL_AGENT_API_KEY` 环境变量（火山方舟平台的 API Key）。

<Tabs items={['pip', '源码预览版']}>
  <Tab value="pip">
    ```bash
    # 稳定版
    pip install veadk-python
    ```
  </Tab>
  <Tab value="源码预览版">
    ```bash
    # 主分支预览版
    pip install git+https://github.com/volcengine/veadk-python.git@main
    ```
  </Tab>
</Tabs>

## 创建一个智能体

通过定义 `Agent` 类即可创建一个简单的对话智能体：

```python
from veadk import Agent

agent = Agent(
    name="root_agent",
    description="A helpful assistant for user questions.",
    instruction="Answer user questions to the best of your knowledge",
    model_name="doubao-seed-1-6-250615",
    model_api_key="...",
)
```

参数说明：

- `name`：智能体名称，在被调用时用于标识；
- `description`：智能体描述，在 A2A 场景下用于表明其基本功能；
- `instruction`：系统提示词，定义智能体的行为与响应方式；
- `model_name`：智能体使用的模型名称；
- `model_api_key`：调用模型所使用的 API Key。

## 运行一个智能体

VeADK 使用 `Runner` 来执行智能体，并提供了高度封装的 `run` 方法用于本地测试：

```python
import asyncio

from veadk import Agent, Runner

agent = Agent(
    name="root_agent",
    description="A helpful assistant for user questions.",
    instruction="Answer user questions to the best of your knowledge",
    model_name="doubao-seed-1-6-250615",
    model_api_key="...",
)
runner = Runner(agent=agent)

response = asyncio.run(runner.run("你好"))
print(response)
```

<Callout type="info">
  更多关于 `Runner` 的用法，请参阅[执行引擎](/cn/docs/framework/runner)章节。
</Callout>
