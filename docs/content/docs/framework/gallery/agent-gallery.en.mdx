# Agent Samples

## Getting Started

Focused on basic concepts and quick onboarding, suitable for developers new to agent development. These features are simple to configure and let you quickly validate ideas.

- Clearly grasp the basic concepts of AI agents (definitions, use cases) and core components (memory module, tool-calling layer, decision engine, etc.);
- The mapping between each component and Volcengine products (e.g., the memory component maps to the Volcengine "Vector Database", tool calling maps to "Function Service");
- Use an AI IDE for vibe coding to implement simple agents that call tools;
- The concept and usage of `sub_agents` in multi-agent systems;
- The basic concepts and simple usage of MCP;
- Key FAQs for AI agent development.
