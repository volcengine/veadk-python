Trusted MCP adds "component identity attestation and verification" on top of the standard MCP protocol and provides end-to-end encrypted communication, building a fully trusted agent runtime. Combined with confidential computing (such as Jeddak AICC) and trusted inference services, it effectively mitigates risks such as untrusted service identity, data tampering, traffic hijacking, and privacy leakage.

VeADK natively supports connecting to TrustedMCP services. You can enable a trusted channel in your tool-call chain to secure outbound communication.

## Trusted Agent

Backed by confidential computing technology (such as Jeddak AICC), you can run agents and large-model services (such as Confidential Doubao) in a trusted environment and, via TrustedMCP plus end-to-end encrypted communication, build a fully trusted agent. TrustedMCP is Volcengine's trusted MCP: it extends the standard MCP protocol with identity attestation and verification between core components, providing end-to-end communication security between components. It addresses security threats in MCP applications such as untrusted service identity, data tampering, traffic hijacking, and data privacy leakage.

With VeADK you can build a trusted agent and call TrustedMCP services through the `tools` mechanism.

## Connection Setup and Prerequisites

1. Have a working TrustedMCP service address ready, e.g. `wss://your-trusted-mcp.example.com`.
2. Prepare the AICC configuration file `./aicc_config.json` (used for the confidential environment and remote attestation); refer to the official sample configuration.
3. Set `x-trusted-mcp: true` in the connection parameters to enable trusted mode.

## Usage

```python title="agent.py"
import asyncio

from veadk import Agent
from veadk.utils.mcp_utils import get_mcp_params
from veadk.tools.mcp_tool.trusted_mcp_toolset import TrustedMcpToolset

mcp_url = "<TrustedMCP server address>"

# 1. Enable the TrustedMCP feature and related configuration
connection_params = get_mcp_params(mcp_url)
connection_params.headers = {"x-trusted-mcp": "true"}

# 2. Initialize the TrustedMcpToolset
toolset = TrustedMcpToolset(connection_params=connection_params)

# 3. Initialize the Agent
agent = Agent(tools=[toolset])

# 4. Run the Agent
response = asyncio.run(agent.run("What's the weather in Beijing?"))

print(response)  # The weather in Beijing is sunny, 25°C.
```

## Options

TrustedMcpToolset supports the following key settings:

- `x-trusted-mcp` (Header)
    - Type: string
    - Default: `true`
    - Description: whether to enable the TrustedMCP feature and the trusted channel.

- `aicc_config` (path)
    - Type: string
    - Default: `./aicc_config.json`
    - Description: path to the AICC configuration file, used for attestation and policy configuration of the confidential computing environment.

> For a sample configuration file and details, see: [TrustedMCP configuration file](https://github.com/volcengine/AICC-Trusted-MCP/blob/main/README.md).
