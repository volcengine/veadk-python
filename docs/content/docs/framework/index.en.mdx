**VeADK** (Volcengine Agent Development Kit) is Volcengine's framework for building AI agents. It composes models, tools, memory, and knowledge bases into reusable building blocks and integrates deeply with the Volcengine ecosystem, so you ship production agents with less code.

```python
import asyncio
from veadk import Agent, Runner

agent = Agent(instruction="You are a helpful assistant.")
print(asyncio.run(Runner(agent=agent).run("Hi, introduce yourself")))
```
