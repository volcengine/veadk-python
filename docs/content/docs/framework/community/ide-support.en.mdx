# System Prompt

We use the `llms.txt` mechanism to provide LLMs and AI IDEs with prompts about VeADK.

## llms.txt

`llms.txt` provides LLMs and AI IDEs with prompts about VeADK. You can feed the content below to your LLM or IDE editor to automatically generate VeADK-based code.

## Trae

### How to Use

1. Download the prompt file we prepared for you: [llms-full.txt](/llms-full.txt)
2. Open the settings page in Trae and find the "Rules" option
3. Copy the file content into your Trae rules file
