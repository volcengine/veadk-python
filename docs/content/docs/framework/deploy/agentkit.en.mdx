---
title: "Deploy to AgentKit"
description: "Use the VeADK CLI to generate and deploy a standard AgentKit project."
---

If you want to generate a standard AgentKit project, you can run the following on the command line:

```bash
veadk agentkit init
```

Or:

```bash
agentkit init
```

After it is generated, you can run `veadk agentkit launch` or `agentkit launch` in the project root directory to deploy the project.

For other detailed steps and instructions, see [Deploy to the AgentKit Platform](https://volcengine.github.io/agentkit-sdk-python/content/2.agentkit-cli/1.overview.html#%E4%B8%89%E7%A7%8D%E9%83%A8%E7%BD%B2%E6%A8%A1%E5%BC%8F).
