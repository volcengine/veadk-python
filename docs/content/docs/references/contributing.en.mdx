## VeADK

<Callout type="info" title="We welcome your contributions to VeADK!" />

### Build and Install from Source

Download the VeADK source code locally and install it.

### Development

Please develop according to the [Google Python Style Guide](https://google.github.io/styleguide/pyguide.html).

### Install `pre-commit`

`pre-commit` runs a series of checks before you commit code, such as code-formatting checks and secret-leak checks. Commits that have not passed `pre-commit` checks will not pass the CI/CD workflow checks on GitHub, which means the code will not be merged.

```bash
pip install pre-commit

pre-commit install .
```

### Wait for Code Review

Your contributions will be processed within 1 to 2 business days.
