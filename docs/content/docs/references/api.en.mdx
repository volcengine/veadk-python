VeADK does not publish an auto-generated API reference site. This page is a quick map of the main `veadk.*` packages, so you can find which module holds a capability and which guide covers it. For per-symbol signatures and docstrings, browse the [PyPI package](https://pypi.org/project/veadk-python/) or the [source on GitHub](https://github.com/volcengine/veadk-python/tree/main/veadk).

## Top-level entry points

The two most common objects are importable directly from `veadk`:

```python
from veadk import Agent, Runner
```

| Object | Description |
| :-- | :-- |
| `Agent` | The core agent class — wraps the model, tools, memory, and knowledge base. See [Create an Agent](/en/docs/framework/agent). |
| `Runner` | The execution engine that runs an agent and manages sessions. See [Runner](/en/docs/framework/runner). |
