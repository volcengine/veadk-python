VeADK 没有自动生成的 API 参考站点。本页把 `veadk.*` 的主要包整理成一张速查地图，帮你快速定位某个能力在哪个模块、对应哪篇指南。需要逐个符号的签名与 docstring 时，请直接看 [PyPI 包](https://pypi.org/project/veadk-python/)或 [GitHub 源码](https://github.com/volcengine/veadk-python/tree/main/veadk)。

## 顶层入口

最常用的两个对象可直接从 `veadk` 导入：

```python
from veadk import Agent, Runner
```

| 对象 | 说明 |
| :-- | :-- |
| `Agent` | 智能体核心类，封装模型、工具、记忆、知识库等能力。参见[构建智能体](/cn/docs/framework/agent)。 |
| `Runner` | 执行引擎，负责运行智能体并管理会话。参见[执行引擎](/cn/docs/framework/runner)。 |
