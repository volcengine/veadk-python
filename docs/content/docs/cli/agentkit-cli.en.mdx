## Overview

The VeADK CLI is compatible with AgentKit commands. You can simply add the `veadk` prefix to AgentKit-related commands, reusing the familiar AgentKit workflow to create, configure, and deploy projects to the Volcengine AgentKit platform.

<Callout type="info">
  Before using the AgentKit-compatible commands, make sure you have installed VeADK and configured your Volcengine access credentials in the `.env` file in your project root.
</Callout>

## Command Reference

| Command | Description |
| :-- | :-- |
| `veadk agentkit init` | Create a new AgentKit project. |
| `veadk agentkit config` | Configure an AgentKit project. |
| `veadk agentkit launch` | Deploy the project to the Volcengine AgentKit platform. |

## Create a Project

Use the `init` command to create a new AgentKit project:

```bash
veadk agentkit init
```

## Configure a Project

Use the `config` command to configure the project:

```bash
veadk agentkit config
```

## Deploy a Project

Use the `launch` command to deploy the project to the Volcengine AgentKit platform:

```bash
veadk agentkit launch
```

## Further Reference

For more AgentKit commands and their full parameter descriptions, see the [AgentKit CLI](https://volcengine.github.io/agentkit-sdk-python/content/2.agentkit-cli/1.overview.html).
